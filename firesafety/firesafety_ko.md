# 개인정보처리방침

본 개인정보처리방침은 모바일 애플리케이션 `소방점검이`(이하 `앱`)에 적용됩니다.

- 시행일: `2026-04-11`
- 앱명: `소방점검이`
- 개인정보처리방침 문의처: icecrao2@icloud.com
- 개인정보처리자: Moonworks

## 1. 개요

앱은 소방점검 업무를 기록하고, 사진·서명·PDF 문서를 생성 및 관리할 수 있도록 설계된 업무용 앱입니다.

앱은 원칙적으로 사용자가 입력하거나 첨부한 데이터를 사용자의 기기 내부에 저장하며, 개발자 서버로 회원정보나 점검 데이터를 전송하거나 별도로 축적하지 않습니다. 다만 사용자가 운영체제의 공유 기능, 출력 기능 또는 외부 앱 연동 기능을 직접 실행하는 경우, 사용자가 선택한 대상에게 문서가 전달될 수 있습니다.

## 3. 정보 처리 목적

앱은 다음 목적 범위에서만 정보를 처리합니다.

- 대상물 등록, 조회, 검색, 수정, 보관
- 점검 생성, 이어쓰기, 반복 점검 재사용
- 불량사항 기록 및 정렬
- 사진 첨부 및 캡션 관리
- 서명 입력 및 저장
- 지적서 PDF 및 점검 요약 PDF 생성
- 생성 문서 저장, 미리보기, 출력, 공유
- 앱 재실행 후 기존 작업 복원

## 4. 정보 수집 및 저장 방식

앱은 현재 확인된 구현 기준으로 다음과 같이 동작합니다.

- 구조화 데이터는 사용자의 기기 내부 `SQLite` 데이터베이스에 저장됩니다.
- 사진, 서명, PDF는 사용자의 기기 내부 앱 저장공간에 파일로 저장됩니다.
- 앱은 점검 데이터 처리의 주된 수단으로 자체 서버 업로드 기능이나 회원 계정 기반 동기화 기능을 제공하지 않습니다.
- 앱 내 데이터는 기본적으로 기기 내에서만 처리되며, 사용자가 직접 공유 또는 출력 기능을 실행한 경우에만 외부 대상에게 전달될 수 있습니다.

## 6. 제3자 제공 및 공유

앱은 사용자가 입력한 점검 데이터를 개발자 서버에 판매하거나 제3자에게 임의 제공하지 않습니다.

다만 아래 경우에는 사용자의 명시적 실행에 따라 데이터가 외부로 전달될 수 있습니다.

- 사용자가 시스템 공유 기능을 통해 메신저, 이메일, 파일 앱 등 다른 앱으로 PDF를 공유하는 경우
- 사용자가 시스템 인쇄 또는 저장 기능을 실행하는 경우
- 사용자가 운영체제 또는 클라우드 백업 기능을 사용하는 경우, 해당 운영체제 또는 클라우드 사업자의 정책에 따라 기기 데이터가 백업될 수 있는 경우

위 경우의 처리는 사용자가 선택한 외부 서비스 또는 운영체제 제공자의 정책에 따릅니다.

## 7. 국외 이전

앱 자체는 점검 데이터를 개발자 서버로 전송하거나 국외 서버로 이전하도록 설계되어 있지 않습니다.

다만 사용자가 외부 서비스로 파일을 공유하거나, 운영체제 또는 클라우드 백업 기능을 사용하는 경우에는 해당 서비스 사업자의 정책에 따라 국외 이전이 발생할 수 있습니다.

## 8. 보유 및 이용기간

앱은 사용자가 입력하거나 생성한 정보를 다음 기준에 따라 보관합니다.

- 로컬 저장 데이터: 사용자가 앱을 삭제하거나 앱 데이터가 제거될 때까지
- 사진, 서명, PDF 파일: 사용자가 앱을 삭제하거나 관련 파일이 제거될 때까지
- 일부 데이터는 앱의 무결성 정책에 따라 즉시 영구 삭제되지 않고 `archive(보관)` 상태로 처리될 수 있습니다.

현재 확인된 구현 기준으로, 대상물과 점검은 일부 경우 영구 삭제 대신 보관 처리될 수 있습니다. 따라서 사용자가 앱 내부에서 삭제를 실행하더라도 일부 정보는 즉시 완전 삭제되지 않고 로컬 기기 내 보관 상태로 유지될 수 있습니다.

개발자는 별도 서버에 사용자 점검 데이터를 보관하지 않으므로, 서버 측 별도 보관기간은 존재하지 않습니다.

## 9. 파기 방법

앱이 처리하는 정보의 파기 방법은 다음과 같습니다.

- 로컬 데이터베이스 데이터: 앱 삭제 또는 앱 데이터 제거 시 운영체제 정책에 따라 삭제될 수 있습니다.
- 사진, 서명, PDF 파일: 앱 삭제, 앱 데이터 제거, 또는 앱 내부 기능에 의한 파일 삭제 시 제거될 수 있습니다.
- 보관 상태로 전환된 데이터는 앱 정책상 즉시 완전 삭제되지 않을 수 있습니다.

## 10. 이용자의 권리와 행사 방법

이용자는 자신의 기기에서 다음과 같은 방법으로 정보 통제를 할 수 있습니다.

- 앱 내에서 대상물, 점검, 문구, 사진 등 수정 또는 일부 삭제
- 앱 권한 설정에서 카메라 또는 사진 접근 권한 철회
- 앱 삭제 또는 운영체제의 앱 데이터 삭제 기능 사용
- 문서 공유 전 내용 검토 후 전송 여부 결정

앱은 현재 별도의 회원 계정 체계를 운영하지 않으므로, 개발자가 원격 서버에서 별도의 회원정보를 조회·정정·삭제하는 방식은 제공하지 않습니다.

## 11. 안전성 확보 조치

앱은 현재 확인된 구현 기준으로 다음과 같은 방식으로 정보를 보호합니다.

- 점검 관련 구조화 데이터는 기기 내부 데이터베이스에 저장
- 사진, 서명, PDF는 기기 내부 앱 저장공간에 저장
- 앱 내부에서 필요한 범위 내에서만 사진, 서명, 문서를 처리
- 사용자가 직접 실행하지 않는 한 점검 데이터를 개발자 서버로 전송하지 않음

다만 사용자의 기기 보안 설정, 운영체제 보안 수준, 백업 설정, 잠금 설정 등에 따라 실제 보호 수준은 달라질 수 있습니다.
